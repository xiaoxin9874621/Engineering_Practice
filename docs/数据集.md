# 数据集

本项目使用以下公开数据集进行 OCR 识别与自动批改功能的验证测试。

## 1. SCUT-HCCDoc — 手写中文文档数据集

👉 https://github.com/HCIILAB/SCUT-HCCDoc_Dataset_Release

- 来源：华南理工大学 HCIILAB 实验室
- 内容：真实手写中文文档图像，涵盖多种书写风格
- 用途：验证系统中文字符 OCR 识别与手写作业批改能力

## 2. EMNIST — 扩展手写字符数据集

👉 https://www.kaggle.com/datasets/crawford/emnist

- 来源：NIST Special Database 19 扩展版本
- 内容：包含手写数字与大小写英文字母共 814,255 张字符图像
- 用途：验证系统英文字符识别与选择题答案判别能力

# 用户操作手册

## 1. 登录

打开 App 后进入登录页。

| 角色 | 用户名 | 密码 |
| --- | --- | --- |
| 教师 | `teacher` | `Demo@123` |
| 学生 | `student` | `Demo@123` |

登录页也提供演示入口。演示入口使用本地样例数据，不依赖后端；正式验证请使用账号密码登录。

## 2. 教师端

### 查看作业

教师登录后进入作业管理页，可查看自己创建的作业列表。作业卡片展示标题、学科、满分、截止时间和状态。

### 创建作业

操作步骤：

1. 点击创建作业。
2. 填写作业标题、学科、班级 ID。
3. 添加多道题目。
4. 每道题填写题型、分值、题目内容、标准答案或关键词。
5. 点击提交到后端。

题型说明：

| 题型 | 输入值 |
| --- | --- |
| 选择题 | 1 |
| 填空题 | 2 |
| 简答题 | 3 |

### 查看学生提交

教师端有两种查看方式：

- 在作业列表中点击某个作业，查看该作业下的学生提交。
- 切换到提交页面，查看自己负责班级内的全部提交。

提交记录展示学生、作业、提交时间、批改状态和总分。点击记录可以进入批改详情。

### 查看批改详情和添加批注

批改详情页展示：

- 学生上传原图。
- OCR 识别结果。
- 总分。
- 每题得分、对错和反馈。
- 教师批注。

教师可以在批注输入框中填写说明并保存，学生端也能看到该批注。

### 查看统计分析

统计分析页展示：

- 总体平均分。
- 总体及格率。
- 批改进度。
- 各班级作业数、提交数、已批改数、平均分、及格率。

教师只能看到自己负责班级的数据。

## 3. 学生端

### 查看待完成作业

学生登录后进入待完成页面。页面显示待完成数量和未提交作业列表。

计算方式：

```text
待完成 = 当前学生所在班级已发布作业 - 当前学生已提交作业
```

提交成功后，该作业会从待完成列表移除。

### 上传作业

操作步骤：

1. 点击待完成作业。
2. 选择拍照或相册。
3. 选择作业图片。
4. 确认图片预览正常。
5. 点击提交作业。

图片建议：

- JPG 或 PNG。
- 内容清晰。
- 题号清楚。
- 不超过 10MB。

推荐作答格式：

```text
1. A
2. 光合作用
3. 两点可以确定一条直线，函数图像也是一条直线。
```

### 查看批改详情

提交后可以进入批改详情页。页面展示：

- 上传原图。
- 当前状态。
- 总分。
- 各题 OCR 识别内容。
- 每题得分。
- 正确、错误或部分正确。
- 批改反馈。
- 教师批注。

如果刚提交时还在识别或批改中，可以稍后刷新或重新进入详情页。

### 查看提交记录和重新提交

底部切换到提交记录，可以看到所有已提交作业。

每条记录支持：

- 查看详情：查看批改结果。
- 重新提交：重新选择图片并覆盖旧提交。

重新提交后，后端会清空旧批改结果，重新执行 OCR 和自动批改。

### 查看个人信息、历史成绩和错题

底部切换到“我的”，页面展示：

- 姓名。
- 学号，也就是用户 ID。
- 账号。
- 班级。
- 已提交次数。
- 已批改次数。
- 平均分。
- 历史成绩。
- 错题查询。

错题列表会展示错误或部分正确的题目，包括作业标题、题号、学生答案、参考答案和反馈。

## 4. 常见问题

### 作业提交后为什么还在待完成里？

下拉刷新或重新进入学生首页。当前版本会根据提交记录自动移除已提交作业。

### 上传后为什么看不到图片预览？

确认使用的是最新 APK。上传页选择拍照或相册后，预览区域应立即显示所选图片。

### OCR 识别和答案对不上怎么办？

尽量让图片中题号清楚，例如：

```text
1.
2.
3.
```

后端已经做了答案对齐处理，会过滤标题、姓名、题型提示等非答案文字，但清晰题号仍然能明显提升效果。

### 真实手机为什么登录失败？

真实手机不能使用 `10.0.2.2`。需要把 Android 后端地址改成电脑局域网 IP，并重新打包 APK。

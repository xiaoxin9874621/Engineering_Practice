## 1. 系统概述

### 1.1 系统简介

手写作业 OCR 识别与自动批改系统是一款基于人工智能技术的教育辅助工具，旨在帮助教师高效批改学生手写作业，减轻教学负担，同时为学生提供及时的学习反馈。

系统采用先进的 PaddleOCR 深度学习模型，能够准确识别手写文字，并结合智能批改算法，实现选择题、填空题、简答题的自动批改。

### 1.2 系统架构

系统由以下组件构成：

- **Android 客户端**: 教师和学生使用的移动端应用
- **Spring Boot 后端**: 业务逻辑处理和数据管理
- **OCR 微服务**: 手写文字识别引擎
- **MySQL 数据库**: 持久化数据存储
- **Redis 缓存**: 提升系统性能

### 1.3 核心功能

| 角色 | 功能模块 | 说明 |
|------|----------|------|
| **教师** | 作业管理 | 创建、发布、管理作业 |
| **教师** | 提交查看 | 查看学生提交记录 |
| **教师** | 批改详情 | 查看自动批改结果，添加批注 |
| **教师** | 统计分析 | 班级成绩统计、批改进度 |
| **学生** | 作业查看 | 查看待完成作业列表 |
| **学生** | 作业提交 | 拍照或相册上传手写作业 |
| **学生** | 批改结果 | 查看批改详情和得分 |
| **学生** | 错题管理 | 查看历史错题记录 |

## 2. 快速入门

### 2.1 环境要求

- **Android 设备**: Android 8.0 (API 26) 及以上
- **网络连接**: 需要连接互联网访问后端服务

### 2.2 登录系统

打开应用后，进入登录页面：

1. **输入用户名和密码**
2. **点击"登录"按钮**

**演示账号（无需注册即可体验）**:

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 教师 | `teacher` | `Demo@123` |
| 学生 | `student` | `Demo@123` |

**快捷登录**: 点击页面底部的"演示教师"或"演示学生"按钮，无需输入账号密码即可进入系统体验完整功能。

### 2.3 注册账号

如果您需要创建新账号：

1. 点击登录页面右下角的"还没有账号？立即注册"
2. 填写以下信息：
   - **真实姓名**: 您的姓名
   - **用户名**: 登录账号（唯一）
   - **密码**: 至少6位
   - **邮箱（可选）**: 用于找回密码
   - **角色**: 选择教师或学生
3. 点击"注册"按钮完成注册

## 3. 教师端功能指南

### 3.1 教师主页

登录教师账号后，进入教师主页，底部导航包含三个标签页：

- **作业管理**: 查看和管理已创建的作业
- **提交记录**: 查看学生提交的作业列表
- **统计分析**: 班级成绩统计和分析

### 3.2 创建作业

1. 在"作业管理"页面点击"创建作业"按钮
2. 填写作业基本信息：
   - **作业标题**: 如"数学周测：一次函数"
   - **学科**: 如"数学"、"语文"等
   - **班级 ID**: 默认为1（演示班级）
3. 添加题目（至少添加一道）：
   - **第N题**: 题目序号自动递增
   - **题型**: 
     - `1`: 选择题（答案精确匹配）
     - `2`: 填空题（答案精确匹配）
     - `3`: 简答题（关键词匹配，逗号分隔多个关键词）
   - **分值**: 该题的满分分数
   - **题目**: 题目内容描述
   - **参考答案/关键词**: 
     - 选择题/填空题：填写标准答案
     - 简答题：填写关键词，用逗号分隔（如"两点,直线,函数"）
4. 点击"添加题目"可添加更多题目
5. 点击"提交到后端"完成作业创建

### 3.3 作业列表

作业管理页面展示所有已创建的作业：

- **作业标题**: 点击可查看该作业的学生提交列表
- **学科**: 显示作业所属学科
- **满分**: 显示作业总分
- **截止时间**: 作业截止提交时间
- **状态**: 草稿/已发布/已结束

### 3.4 查看学生提交

1. 在作业列表中点击某个作业，进入该作业的提交列表页面
2. 列表显示：
   - 提交学生编号
   - 提交时间
   - 状态（等待识别/识别中/批改中/批改完成）
   - 得分（批改完成后显示）
3. 点击"查看批改"可查看详细批改结果

### 3.5 批改详情

批改详情页面展示：

- **上传图片**: 学生提交的作业图片
- **状态**: 当前处理状态
- **总分**: 自动批改得出的总分
- **各题详情**: 每题的批改结果
  - **题号**: 题目序号
  - **学生答案**: OCR识别出的学生手写答案
  - **得分**: 该题得分/满分
  - **结果**: 正确/错误/部分正确
  - **反馈**: 自动生成的批改反馈

**添加批注（教师专属）**:
1. 在页面底部输入框中输入批注内容
2. 点击"保存批注"按钮
3. 批注会显示在页面上方的批注列表中

### 3.6 统计分析

统计分析页面展示班级整体学习情况：

- **平均分**: 所有班级的平均分数
- **及格率**: 已批改作业的及格比例
- **批改进度**: 已批改/总提交数量

**班级对比**:
- 显示每个班级的详细统计：
  - 作业数量
  - 提交次数
  - 已批改次数
  - 平均分进度条
  - 及格率进度条

### 3.7 退出登录

点击页面右上角的"退出"按钮，系统将清除登录状态并返回登录页面。

## 4. 学生端功能指南

### 4.1 学生主页

登录学生账号后，进入学生主页，底部导航包含三个标签页：

- **我的作业**: 待完成作业列表
- **提交记录**: 已提交作业历史
- **个人中心**: 个人信息和学习数据

### 4.2 待完成作业

"我的作业"页面展示当前待完成的作业：

- **待完成数量**: 显示还有多少作业未提交
- **作业列表**: 显示所有已发布且未提交的作业
  - **作业标题**: 点击可进入提交页面
  - **学科**: 作业所属学科
  - **满分**: 作业总分
  - **截止时间**: 作业截止时间

### 4.3 提交作业

1. 在待完成作业列表中点击某个作业，进入提交页面
2. **选择图片**:
   - **拍照**: 使用相机拍摄手写作业
   - **从相册选择**: 从手机相册中选择已拍摄的作业图片
3. **预览图片**: 选择图片后可在页面中预览
4. **提交作业**: 点击"提交作业"按钮上传图片
5. 提交成功后，系统会自动跳转到批改详情页面

### 4.4 查看批改结果

提交作业后，进入批改详情页面：

- **等待处理**: 显示"等待识别"状态，系统正在处理
- **实时刷新**: 系统会每2秒自动刷新一次状态
- **查看结果**: 批改完成后显示：
  - 上传图片预览
  - 总得分
  - 每题的详细批改结果
  - 教师批注（如果有）

### 4.5 提交记录

"提交记录"页面展示所有已提交的作业：

- **作业标题**: 点击可查看批改详情（仅批改完成的可查看）
- **提交时间**: 提交作业的时间
- **状态**: 当前处理状态
- **得分**: 批改完成后显示
- **重新提交**: 点击"重新提交"按钮可重新提交该作业

### 4.6 个人中心

个人中心页面展示：

- **个人信息**:
  - 姓名
  - 学号
  - 账号
  - 班级
- **学习概览**:
  - 已提交次数
  - 已批改次数
  - 平均分
  - 错题数量
- **历史成绩**: 按时间倒序显示所有提交记录及得分
- **错题查询**: 显示所有答错或部分正确的题目
  - 作业名称和题号
  - 学生答案和参考答案对比

### 4.7 退出登录

点击页面右上角的"退出"按钮，系统将清除登录状态并返回登录页面。

## 5. 常见问题

### 5.1 登录问题

**Q: 忘记密码怎么办？**

A: 当前版本暂不支持密码找回功能，请联系系统管理员重置密码。演示账号密码固定为 `Demo@123`。

**Q: 登录失败提示"服务器无响应"？**

A: 请检查网络连接是否正常。如果使用演示模式登录，无需网络连接即可体验。

### 5.2 作业提交问题

**Q: 拍照后图片不显示？**

A: 请检查相机权限是否已授予。在手机设置中找到"作业批改助手"，确保相机权限已开启。

**Q: 提交作业失败？**

A: 
1. 检查网络连接是否正常
2. 确保图片格式正确（支持 JPG、PNG、WEBP）
3. 图片大小不能超过10MB

**Q: 可以重新提交作业吗？**

A: 可以。在"提交记录"页面，找到需要重新提交的作业，点击"重新提交"按钮即可。重新提交后，系统会覆盖旧提交并重新进行OCR识别和批改。

### 5.3 OCR识别问题

**Q: 识别结果不准确怎么办？**

A: 
1. 确保手写字迹清晰工整
2. 拍摄时光线充足，避免阴影
3. 图片中不要包含无关内容
4. 确保作业纸张平整，没有褶皱

**Q: 简答题如何评分？**

A: 简答题采用关键词匹配评分。教师在创建作业时设置关键词，系统会统计学生答案中包含的关键词数量，按比例计算得分。

### 5.4 批改问题

**Q: 批改需要多长时间？**

A: 正常情况下，批改过程需要5-30秒，具体时间取决于图片大小和服务器负载。提交后系统会自动刷新状态，无需手动刷新。

**Q: 自动批改有误怎么办？**

A: 当前版本暂不支持人工修改自动批改结果。教师可通过添加批注的方式对学生进行补充说明。

### 5.5 其他问题

**Q: 如何切换角色？**

A: 退出当前账号，使用对应角色的账号重新登录即可。

**Q: 可以在多台设备上登录同一账号吗？**

A: 可以，但为了账号安全，建议在同一时间只在一台设备上登录。

## 6. 附录

### 6.1 题型说明

| 题型代码 | 题型名称 | 批改方式 | 说明 |
|----------|----------|----------|------|
| 1 | 选择题 | 精确匹配 | 学生答案与标准答案完全一致即为正确 |
| 2 | 填空题 | 精确匹配 | 学生答案与标准答案完全一致即为正确 |
| 3 | 简答题 | 关键词匹配 | 根据包含的关键词数量按比例得分 |

### 6.2 状态码说明

| 状态码 | 状态名称 | 说明 |
|--------|----------|------|
| 0 | 等待识别 | 作业已提交，等待OCR识别 |
| 1 | 识别中 | OCR服务正在处理图片 |
| 2 | 识别完成 | OCR识别完成，等待批改 |
| 3 | 批改中 | 系统正在进行自动批改 |
| 4 | 批改完成 | 自动批改已完成 |
| 5 | 处理失败 | 识别或批改过程中发生错误 |

### 6.3 得分结果说明

| 结果码 | 结果名称 | 说明 |
|--------|----------|------|
| 1 | 正确 | 答案完全正确，得满分 |
| 0 | 错误 | 答案完全错误，得0分 |
| 2 | 部分正确 | 简答题包含部分关键词，按比例得分 |
